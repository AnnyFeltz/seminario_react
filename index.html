Seminário: Introdução ao React

1. Introdução ao React (3 minutos)

Responsável: Pessoa 1

Texto do Slide:

  • O que é React?
    • Biblioteca JavaScript criada pelo Facebook.
    • Foco: interfaces de usuário dinâmicas e responsivas.
  • Por que React?
    • Componentes reutilizáveis.
    • Alta produtividade.
    • Interatividade sem recarregar a página inteira.
  • Grandes empresas que usam React: Facebook, Instagram, Netflix.

Texto para falar:

React é uma biblioteca JavaScript criada pelo Facebook, voltada para a criação de interfaces de usuário que são dinâmicas e responsivas. Ele é muito usado porque permite reutilizar componentes, o que torna o desenvolvimento mais organizado e produtivo. Além disso, com React, a página atualiza automaticamente partes específicas quando os dados mudam, sem precisar recarregar tudo. Um exemplo de quem usa React são empresas como Facebook, Instagram e Netflix.

Texto para falar:

Um dos conceitos principais do React são os componentes. Eles são como peças de Lego: cada componente é uma parte independente da interface que pode ser criada e reutilizada. Por exemplo, aqui temos um componente simples chamado Cabecalho, que retorna um título na tela. Outro conceito importante é o JSX, que é uma sintaxe especial do React. Ele mistura HTML e JavaScript no mesmo código, o que facilita a leitura e escrita. Por exemplo, aqui temos o JSX que cria um elemento HTML diretamente dentro do código JavaScript. Com JSX, tudo fica mais intuitivo.
